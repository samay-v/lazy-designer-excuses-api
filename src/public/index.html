Lazy designer excuses - API documentation

I was too lazy to create a truly awesome thing with this data so I made an API for you too build something better. Use this API to create a slack bot that sends a random excuse on your design stand-up everyday, create wallpapers with an excuse for each day of the year, or just play with it and make me spend $$$ on server costs, idk what you're into.

The documentation under here is the bare minimum I could do, for a better view at the how the api works run this collection on postman...

GET /random

    Get a random designer excuse

    https://api.lazydesignerexcuses.com/random

GET /random/{tag}

    Get a random excuse from a perticular tag

    https://api.lazydesignerexcuses.com/random/{{tag}}

GET /tags

    Get all unique tags

    https://api.lazydesignerexcuses.com/tags

GET /all

    Get all the excuses togeather

    https://api.lazydesignerexcuses.com/all

GET /id

    Get an excuse by its corresponding id

    https://api.lazydesignerexcuses.com/{{id}}

GET /total

    Get the total number of excuses present.

    https://api.lazydesignerexcuses.com/total
